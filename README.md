#需求目标单张2寸照片加入信息变成4张
###注意：在windows下，代码文件本身的编码必须是ansi格式的才能避免乱码。数据库（utf-8的字符集）执行的时候要写个query：set names gbk
ANSI 在中文操作系统中对应的字符集就是 gbk
单个程序逻辑很简单
* 遍历目标文件夹下所有目标文件，存到数组中
* 从数据库中拿到所有的目标文件相应信息，存到数组中
* 根据目标文件相应的信息，在当前脚本文件的同级目录中建立文件夹存放图片处理结果
* gd库中的图片处理函数写的图片处理逻辑
